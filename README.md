## 4주차 과제

  - 영화, 드라마에서의 스킨십(키스신 등) 여부를 알 수 있는 앱
  
    - 부모님이나 친구와 같이 영화나 드라마를 보는 중에 민망한 장면이 나왔던 경험이 한 번쯤은 있을 것이다.
    
      그런 장면이 나오는지의 여부와 나온다면 언제쯤에 나오는지를 알 수 있다면 무엇을 볼지 고르기가 수월해질 것 같다.
